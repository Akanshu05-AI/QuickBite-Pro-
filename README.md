# QuickBite Pro | Smart Mess Management System

QuickBite Pro is a modern, responsive web application designed to streamline university mess operations. It features a sleek UI for students to book meals, track health/calories, and manage their digital wallet, alongside a hidden administrative dashboard for real-time attendance and crowd monitoring.

## ✨ Key Features

- **Smart Dashboard**: Real-time clock, dynamic meal greetings, and automated crowd status indicators.
- **Meal Booking System**: Book Breakfast, Lunch, Snacks, or Dinner with automated price calculation and balance checks.
- **Digital Wallet**: Integrated wallet system with transaction history (Credit/Debit) and one-click recharge.
- **Health & Nutrition**: Automated calorie tracking based on selected meals and weekly balance summaries.
- **Admin Hub**: A hidden "Enterprise Admin Hub" with live metrics, revenue tracking, security breach logs, and CSV data export.
- **QR Code Generation**: Dynamic QR code generation for entry verification.
- **Dietary Filters**: Quickly filter menus by Veg, Non-Veg, Jain, or High Protein.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Custom Variables, Flexbox, Grid)
- **Logic**: Vanilla JavaScript (ES6+)
- **Storage**: Browser LocalStorage (Persistence without a backend)
- **External Libraries**: [QRCode.js](https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js) for meal pass generation.

## 🚀 Getting Started

Since this is a client-side application, no installation is required!
